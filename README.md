# ZY Diary

ZY Diary的主要理念在于通过记录、打卡、复盘、反思等活动对自己的生活和经济支出进行管理与记录，在数据上力求本地化~~(这样就算软件跑路或者停止更新仍旧可以保存旧版本的正常运行)~~，同时加入了修仙系统，希望给使用者更好的正向反馈。与此同时，在软件体量越来越大的当下，本软件轻量化设计尽量少吃存储，这也正呼应了我们的初衷：用细微的行动，推动用户实现自我的壮大。

*用省下的存储做自己喜欢的事情喵！^-^*

## 项目简介

这个项目希望解决的不是单纯的记录，而是“如何把生活中的记录、行动、消费和反思，慢慢沉淀成可回看、可追踪、可恢复的长期轨迹”。即用细小的记录描摹自身的成长轨迹，让自己的未来有迹可循。

在当前发行版本中，ZY Diary 已经具备：

- 每日记录与往日复盘分离的使用流程
- 目标创建、打卡、月历反馈与奖励惩罚转盘
- 消费记录、预算系统、固定消费与月份切换
- 周报月报与阶段洞察
- 用户名、主题切换、帮助中心与首页首次引导卡
- 轻提醒与原生本地通知能力
- 本地数据导出、导入、恢复与跨平台存储支持
- Windows 安装包与 Android APK 发布能力

## 核心功能

### 1. 每日记录

- 记录今天最重要的完成事项、阻碍和下一步
- 支持情绪、精力、专注度状态记录
- 配合首页开屏问候形成稳定的每日进入节奏

- 独立页面查看最近六条历史记录

- 支持创建与编辑日常目标
- 支持按月打卡与独立日志记录
- 已完成日期可在日历中直观看到状态反馈
- 支持自定义奖励转盘与惩罚转盘内容

### 4. 账本与预算

- 记录日常消费并按类别统计
- 支持月预算与分类预算
- 支持固定消费项目
- 支持按月份查看账本、预算执行情况和图表变化

- 自动汇总周报与月报

### 6. 个性化与帮助

- 支持用户名设定
- 支持 `晨雾米金` 与 `夜墨青金` 两套主题
- 支持帮助中心页面

- 浏览器环境支持本地存储
- 桌面版支持文件目录存储
- 安卓支持应用私有目录与设备文档目录

- Windows 应用标识：`com.zydiary.desktop`

## 安装方式

### Windows

直接运行安装包：

```text
dist/ZY-Diary-Setup-1.0.1.exe
```

安装器支持用户自行选择安装目录。

### Android

直接安装签名后的 release APK：

```text
android/app/build/outputs/apk/release/app-release.apk
```

首次安装时，请根据系统提示授予应用必要权限，以便本地通知和文档目录存储正常工作。

## 推荐使用流程

新用户建议按以下顺序开始：

1. 进入“档案”页设置用户名、主题和提醒时间
2. 进入“目标”页创建 2 到 5 个日常目标
3. 进入“记录”页写下第一条正式记录
4. 进入“帮助”页了解页面结构与数据说明
5. 在后续使用中通过“报告”页观察周期变化

## 数据安全说明

- 桌面版默认会把数据保存到本地目录，并支持查看和切换存储位置
- 安卓版支持应用私有目录和设备文档目录两种模式
- 建议在升级版本、迁移设备或做大规模结构调整前先导出一次 JSON 备份
- 项目内已接入恢复快照机制，可在“档案”页查看当前恢复信息

## 开发与构建

### 桌面版开发

```bash
npm install
npm run dev
```

### 打包 Windows 安装包

```bash
npm install
npm run dist
```

### 安卓资源同步

```bash
npm install
npm run android:prepare
npm run android:sync
```

### 打开 Android Studio

```bash
npm run android:open
```

### 构建 Android Release APK

```bash
cd android
./gradlew assembleRelease
```

## 目录说明

- `js/`：源码入口与页面模块
- `app.js`：浏览器和静态页面实际运行入口
- `electron/`：桌面版主进程与预加载桥
- `android/`：Capacitor Android 原生工程
- `build/`：图标和安装器资源
- `dist/`：Windows 打包产物
