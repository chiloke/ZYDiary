ZY Diary

ZY Diary 是一套面向长期生活管理的个人系统，围绕每日记录、往日复盘、目标打卡、消费账本、周期报告、提醒机制与本地数据管理展开，提供桌面版与安卓版双端体验。

当前版本1.0.1
应用名称ZY Diary
Windows 应用标识com.zydiary.desktop
Android 包名com.zydiary.mobile

项目定位

这个项目希望解决的不是“如何多记一点东西”，而是“如何把生活中的记录、行动、消费和反思，慢慢沉淀成可回看、可追踪、可恢复的长期轨迹”。

核心功能

  • 每日记录：记录完成事项、阻碍、下一步，以及情绪、精力、专注度。
  • 往日复盘：查看最近六条记录，并通过日历回溯历史内容。
  • 目标系统：支持创建目标、按月打卡、自定义奖励与惩罚转盘。
  • 账本预算：支持消费记录、月预算、分类预算、固定消费与月份切换。
  • 周期报告：自动生成周报和月报，汇总目标、消费、灵气和复盘变化。
  • 个性化设置：支持用户名、主题切换、帮助中心与首页首次引导卡。
  • 数据管理：支持本地存储、导出备份、导入恢复、快照查看与跨平台存储模式。

平台发布信息

  • Windows 安装包：dist/ZY-Diary-Setup-1.0.1.exe
  • Android Release APK：android/app/build/outputs/apk/release/app-release.apk

推荐使用流程

  1. 进入档案页设置用户名、主题和提醒时间。
  2. 进入目标页创建 2 到 5 个日常目标。
  3. 进入记录页写下第一条正式记录。
  4. 进入帮助页了解页面结构、提醒机制和数据说明。
  5. 后续通过报告页持续观察自己的周期变化。

数据安全说明

  • 桌面版支持本地目录存储，并可查看和切换数据目录。
  • 安卓支持应用私有目录和设备文档目录两种模式。
  • 建议在升级版本、迁移设备或做大规模调整前先导出一次 JSON 备份。
  • 项目已接入恢复快照机制，可在档案页查看恢复信息。

开发与构建

桌面版开发：

npm install
npm run dev

打包 Windows 安装包：

npm install
npm run dist

同步安卓资源：

npm install
npm run android:prepare
npm run android:sync

构建 Android Release APK：

cd android
./gradlew assembleRelease
